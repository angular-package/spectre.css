Introduction

Spectre.css is a lightweight, responsive and modern CSS framework for faster and extensible development.

Spectre provides basic styles for typography and elements, flexbox based responsive layout system, pure CSS components and utilities with best practice coding and consistent design language.

Install Spectre.css

Spectre.css is modified and maintained by the angular-package. Click on the link to see the demonstration created on the @angular.io with modules and components.
